    NCBI API Key (Optional - for faster searches)

    Without an API key: 3 requests/second limit (may fail with many topics).

    With a free API key: 10 requests/second limit (much faster, more reliable).

    To get a free key:

    1. Go to NCBI account page and sign in (or create account)
    2. Click your username (top right) → "Account Settings"
    3. Scroll to "API Key Management" → Click "Create an API Key"
    4. Copy the key and paste it below

    How to Use

    Boolean AND Search: Your keyword is combined with each selected topic. Example: "chronic headaches" + Pediatrics, Cannabis = TWO searches: (chronic headaches AND pediatrics) and (chronic headaches AND cannabis).

    Results Grouped by Topic: Each topic gets its own section showing relevant articles.

    Filters Applied to All Searches: Date range, language, humans-only, and article type filters apply to every search automatically.

    Save Articles: Click "Save to Archive" on any result to store it permanently.
